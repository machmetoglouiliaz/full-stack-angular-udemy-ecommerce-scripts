## 📂 How to Use

1️⃣ **Download the scripts**  
- You can click on each file individually, then select the three-dot menu (⋮) at the top-right and choose **Download**.
- Alternatively, click the **<> Code** button and select **Download ZIP**.

2️⃣ **Run the scripts**  
- Execute the SQL scripts exactly as shown in the course videos.

3️⃣ **Set up product images**  
- Place your product images in the directory: `./public/images/products`  
  *(instead of `./src/assets/images/products` as shown in earlier course versions)*

4️⃣ **Run your application**  
- Your app should now run properly with images displaying as expected.
